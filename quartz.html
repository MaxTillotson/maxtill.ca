Quartz

Project Brief

The concept behind this project was to create our own usable serif font. With full creative control, the plan was to create a geometric, mono weight, serif font with an extra serif.

Sketches

After countless scribbles of letters, a theme was settled on. The font was to be named Quartz. Not only does it use some of the more unique letters in the font, but the name stuck with the creative process behind it.
